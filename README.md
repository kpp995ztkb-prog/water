# 水波交互 🌬️

摄像头 + 食指手势驱动的实时水面互动。伸出食指轻轻拂过，画面（你本人的镜像）会像一池春水般荡漾开来——波纹自然扩散、遇边软反射、彼此干涉、缓缓平息，配上焦散高光与雨天般的冷暖微光。

灵感来自「一池野春水 · 下雨天 · 在家 Relax」的治愈氛围。


本版改用真正的 **GPU 高度场水面模拟**：双缓冲 framebuffer ping-pong + 波动方程，每帧在低分辨率网格上演算波动，再用高度场法线对摄像头画面做折射。质感更接近真实水面。

## 运行

摄像头需要安全上下文（`localhost` 或 `https`）。在项目目录下：

```bash
python3 -m http.server 8000
```

然后浏览器打开 `http://localhost:8000`，允许摄像头权限即可。

- 主交互：MediaPipe 手势模型，追踪食指尖（首次需联网加载模型）
- 兜底：鼠标 / 触摸拂过水面（桌面调试用）
- 纯视觉，无声音

## 技术

- 原生 WebGL，无框架，单页三文件（`index.html` / `styles.css` / `script.js`）
- 高度场模拟需要浮点渲染目标（`FLOAT` 或 `HALF_FLOAT`），启动时自动探测并降级提示
- 调试：控制台 `window.waterDebug`
